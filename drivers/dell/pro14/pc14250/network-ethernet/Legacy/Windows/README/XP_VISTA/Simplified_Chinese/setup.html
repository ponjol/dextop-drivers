驱动程序安装

安装驱动程序时，请按照下列步骤：
  1. 执行安装程序文件 "setup.exe"
  2. 按照安装程序中显示的指示说明进行安装


注意:

    
  • 在支持即插即用的Windows版本上，系统会自动搜索、识别和设置网卡。 请取消操作系统自动安装驱动程序的所有进程。 
  • 安装驱动程序时，您必须使用系统管理员权限。


手动安装驱动程序

手动安装驱动程序是指安装驱动程序时不使用Setup.exe程序进行安装，而是通过INF安装文件进行驱动程序安装。

手动安装时请开启设备管理器，展开网络适配器设备，选择Realtek RTL81XX Ethernet NIC网卡后，按下右键， 执行更新驱动程序，选择从列表或者指定位置安装(高级)，并按对话框提示指定INF安装文件的路径进行安装。


注意：

  • 手动安装驱动程序是属于高级使用者的一种安装方式， 建议一般使用者采用setup.exe来安装驱动程序。


无信息安装

无信息安装是指安装驱动程序的过程中，不会显示对话框在屏幕上一级无需使用者介入操作的背景安装。 若需进行无信息安装或者卸载，可在命令行窗口下，执行下列命令

无信息驱动程序安装 

    C:>%driver_path%\setup.exe /s

无信息驱动程序卸载 
    C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"

故障排除

若在安装驱动程序软件时遇到任何问题，请重新仔细阅读以上安装说明以及故障排除章节，并且再尝试重新安装一次。 若问题仍未改善，可至本公司网站浏览下载最新版的驱动程序重新安装，或与本公司技术服务部门联络。
